# Proyecto Web - Barbería Kevin

Este es un sitio web para la Barbería Kevin, un salón de belleza especializado en cortes de cabello y cuidado de barba.

## Descripción del Proyecto

Este proyecto consta de un sitio web estático desarrollado con HTML, CSS y JavaScript. Utilizamos tecnologías como [Swiper](https://swiperjs.com/) para implementar un slider de imágenes y [Google Maps Embed API](https://developers.google.com/maps/documentation/embed/get-started) para mostrar la ubicación del salón en el sitio.

## Contenido del Repositorio

El repositorio contiene los siguientes archivos y directorios:

- `index.html`: Página principal del sitio.
- `style.css`: Archivo CSS que contiene los estilos para el sitio.
- `reset.css`: Archivo CSS para restablecer los estilos predeterminados del navegador.
- `imagenes/`: Directorio que contiene las imágenes utilizadas en el sitio.
- `diferenciales/`: Directorio que contiene imágenes y contenido relacionado con los diferenciales del salón.
- `js/script.js`: Archivo JavaScript para funcionalidades adicionales del sitio.
- `README.md`: Este archivo que proporciona información sobre el proyecto.
